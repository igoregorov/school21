# 🚀 DevEnv Setup: Среда разработки для студентов (by scratcal)

Этот скрипт превращает стандартную консоль в мощный и удобный инструмент для разработки на C. Никакой ручной настройки, никаких вопросов — просто чистая продуктивность. 

> ⚠️ **Внимание:** Скрипт перезаписывает `~/.bashrc` и `~/.vimrc`. Убедитесь, что вы готовы к изменениям (или сделайте бэкап вручную, если вы не из школы бескомпромиссных решений).

## 🛠 Что внутри?

### 1. Умное и красивое приглашение (Prompt)
Забудьте о длинных, нечитаемых путях. Ваше приглашение теперь показывает:
- Имя пользователя и хост (`user@host`)
- **Сокращенный путь** (например, `projects/my_c_code` вместо `/home/user/projects/my_c_code`)
- **Текущую Git-ветку** (если вы в репозитории), чтобы вы всегда знали, в каком пространстве находитесь.

### 2. Полезные алиасы и функции
Экономим нажатия клавиш:
- `ll`, `la`, `l` — улучшенные варианты `ls` с цветами и деталями.
- `rm`, `cp`, `mv` — автоматически с флагом `-i` (защита от случайного удаления).
- `mkcd <имя>` — создает директорию и сразу переходит в неё (`mkdir` + `cd`).
- `findfile <часть_имени>` — быстрый поиск файлов по имени в текущей директории и ниже.

### 3. `crun`: Компилируй и запускай в одно действие
Забудьте про раздельные команды `gcc` и `./a.out`. 
Просто напишите:
```bash
crun main
# или
crun main.c
```

**Что делает `crun`:**
1. Ищет файл `.c` в текущей папке или в папке `src/`.
2. Компилирует его со строгими флагами: `-Wall -Wextra -Werror` (компилятор не простит ни одного предупреждения!).
3. Если компиляция успешна — сразу запускает программу.
4. **Автоматически удаляет** бинарный файл после завершения, не засоряя рабочую директорию мусором.

### 4. `chck_style`: Жесткий контроль код-стайла
Проверяет ваш код на соответствие стандартам с помощью `clang-format`.
```bash
chck_style main.c
```
**Что делает `chck_style`:**
- Запускает `clang-format` в режиме сухой проверки (`-n`).
- Если есть нарушения, показывает **подробный diff**, подсвечивая, что именно не так.
- Подсказывает команду для автоматического исправления (`clang-format -i main.c`), но предупреждает об ответственности.

### 5. Настроенный Vim (`.vimrc`)
- Включена подсветка синтаксиса и автоотступы.
- Поддержка истинных цветов (`termguicolors`).
- Приятная цветовая схема для строк, комментариев, чисел и ключевых слов (в стиле современных IDE).

## ⚡️ Быстрый старт

1. Скачайте скрипт:
   ```bash
   curl -O https://github.com/igoregorov/school21/blob/master/env_setting.sh
   ```
2. Сделайте его исполняемым и запустите:
   ```bash
   chmod +x env_setting.sh
   ./env_setting.sh
   ```
3. Перезагрузите конфигурацию оболочки:
   ```bash
   source ~/.bashrc
   ```
